[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Tkinter](https://img.shields.io/badge/GUI-Tkinter-orange.svg)](https://docs.python.org/3/library/tkinter.html)

Простое и удобное приложение для управления личным бюджетом с графическим интерфейсом. Помогает отслеживать доходы и расходы, анализировать траты по категориям и сохранять историю операций.

## Возможности

- **Управление доходами и расходами** - быстрое добавление операций
- **Категоризация трат** - продукты, транспорт, жилье и другие категории
- **Визуализация данных** - наглядные графики распределения расходов
- **Сохранение в CSV** - все данные хранятся в удобном формате
- **История операций** - просмотр и удаление записей
- **Простой интерфейс** - интуитивно понятное управление

### Требования

- Git (опционально)
